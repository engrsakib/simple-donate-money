What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a Programming interface for web documents. The DOM representing the strcture of a web pages as a hierarchical tree of objects. Each node has three corresponds to different components of the document, such as elements, attribute and text. The DOM allows developers to dynamically interact with and manipulate the content and structure of a webpage using various programming languages, most notably JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, you can use various methods provided by the DOM API, primarily in JavaScript. Here are some common methods:

  • getElementsByClassName: Selects all elements with a specified class name
  • getElementsByTagName: Selects all elements with a specified tag name
  • querySelector: Selects the first element that matches a specified CSS selector.
  • querySelectorAll: Selects all elements that match a specified CSS selector

    Event delegation is a technique in the context of the DOM that allows you to manage a higher level in the DOM hierachy rather than on individuals elemnts. when an events occurs on child elemnt, it bubbles up to the parent element. By placing the event listener on the parent.

    How do you manipulate an element's attributes and styles using the DOM?

    You can manipulate an element's attributes and styles using the DOM through JavaScript. Here’s how you can do both:
    Manipulating Attributes

  • Getting an Attribute: You can retrieve the value of an attribute using the getAttribute method.
  • Setting an Attribute: You can set or update an attribute using the setAttribute method.
  • Removing an Attribute: To remove an attribute, use the removeAttribute method.
    • Manipulating Styles
  • Getting a Style Property: You can access a specific CSS property using the style property.
  • Setting a Style Property: To change the style of an element, directly assign a value to a CSS property.
  • Adding Multiple Styles: You can also add multiple styles at once using a loop or by directly setting properties.

    • How do I handle events in the DOM?

    You can handle events, like clicks or key presses, by attaching event listeners to elements. This lets you define specific actions that should occur when a user interacts with the webpage, enhancing the interactivity of your site.
